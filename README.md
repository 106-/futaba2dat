# futaba2dat
ふたば☆ちゃんねるを2ch専用ブラウザで読めるように変換するプログラム

## 必要なもの
    # pip install bottle
    # pip install beautifulsoup4

## dockerhubにもあります
[https://hub.docker.com/r/ubiq/futaba2dat/](https://hub.docker.com/r/ubiq/futaba2dat/)

    # docker pull ubiq/futaba2dat
    # docker run --name futaba2dat -p 80:80 -d ubiq/futaba2dat
